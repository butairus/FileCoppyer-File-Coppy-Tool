このフォルダを解凍したら、中に"FileCoppyer.exe"が入っているのでこれを起動。</br>
事前にコピーぺしたいファイルを入れておくフォルダーを作っておく。</br>
コピー内容の入っているフォルダを選択を押して上のフォルダーを選択。</br>
事前にコピーしたファイルをペーストしたいフォルダを作っておく。</br>
コピー先のフォルダを追加を押して上のフォルダを選択。</br>
フォルダは多分約無限に追加できます😅その辺わからん</br>
中央のコピーボタンを押すとコピー先フォルダーにコピーしたいファイル入りのフォルダ内のすべてが、コピーされます。</br>
同じ名前のファイルは上書き保存されるので注意です。</br>
</br>
-----各種曲操作ボタンの説明------</br>
左上から順に</br>
・セットをインポートボタン（セットをエクスポートボタンで作った.texファイルを読み込み、フォルダを自動で選択します。）</br>
・セットをエクスポートボタン（現在選択されているフォルダーすべてを記憶した.texを作成します。）</br>
・コピー内容の入っているフォルダを選択ボタン（コピー先フォルダー内にコピーしたいファイルの入っているフォルダを選択します。）</br>
・コピーボタン（コピー内容の入っているフォルダを選択ボタンで選択したフォルダ内のファイルすべてを、</br>
コピー先のフォルダを追加ボタンで選択したフォルダ内にコピーします。）</br>
・コピー先のフォルダを追加　（コピーしたいファイルを貼り付けたいフォルダを選択し、</br>
コピー先フォルダーリストに追加します。約無限個追加できます。）</br>
</br>
例えばスマホとPCに音楽をいちいちコピペするのが面倒な場合、大変役立ちます。</br>
